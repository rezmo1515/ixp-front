## 🚀 نحوه استفاده

### 1. **شروع برنامه**
```bash
cd /home/mohammadreza/projects/front-ixp
npm run dev
```

### 2. **دسترسی به صفحات**
- **ورود/ثبت نام:** `http://localhost:5173/login`
- **تست Ping:** `http://localhost:5173/ping`
- **بررسی DNS:** `http://localhost:5173/dns`
- **WHOIS:** `http://localhost:5173/whois`
- **پیام‌ها:** `http://localhost:5173/messages`
- **نقش‌ها:** `http://localhost:5173/roles` (فقط مدیران)

### 3. **تست مجوزها**
```typescript
const { hasPermission } = useAuth()

// بررسی مجوز
if (hasPermission('roles.manage')) {
  // دسترسی دارد
}
```

### Q: چگونه ماژول جدید اضافه کنم؟

### Q: چگونه مجوز جدید اضافه کنم؟

**A:** تمام صفحات قبلاً فارسی شده‌اند، فقط نیاز است translate کنید

### Q: Backend API چگونه باید باشد؟
